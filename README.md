# CodeAlpha Data Redundancy Removal System

This project is a cloud-based system designed to detect and remove duplicate data entries from a database.
The system validates incoming data, checks for redundancy, and ensures that only unique and verified records are stored in the cloud database.

Features:

* Automatic duplicate data detection
* Validation mechanism for new data entries
* Prevention of redundant records in the database
* Improved database accuracy and efficiency
* Cloud-ready architecture

Technologies Used:
Python
Flask
MySQL
SQL Queries
HTML/CSS
Cloud Deployment
